TRex Emulation Python API
==============================================

This site covers the Python API of TRex emulation and explains how to utilize it to your needs.
To understand entirely how the API works, check out the `emu doc <https://github.com/cisco-system-traffic-generator/trex-core/wiki>`_ under the documentation section of TRex website.

This website will explain the basics of EMUClient and EMUProfile. As in other TRex modes, the profile describe the wanted actions we want to apply on the server and the client is used in order to send the profile and communicate with the server.
